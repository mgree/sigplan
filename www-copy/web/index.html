TMPL: Topic Modeling in PL

Tracking the flow of ideas through the programming languages literature

Michael Greenberg, Kathleen Fisher, and David Walker


Check out our SNAPL paper, titled Tracking the flow of ideas through the programming languages literature. We've put up some of the figures from the paper (and more recent work) in a larger format. We've also built a tool to help search for related work using the topic models we built. You can also look at our analysis of POPL 2015.


Upload a PDF and we'll give you the most closely related papers according to the topic model, number of topics, and distance metric selected.


Models
The type of corpus used. The abstract corpus contains one "document" for every paper from ICFP (1996-2014), OOPSLA (1986-89,1991-2014), PLDI (1988-2014), and POPL (1973,1975-2015). Each document contains the paper title, author list, and abstract. The full-text corpus contains one "document" for each of the 2,257 POPL and PLDI papers that have appeared as of February 2015.
# of Topics
The desired number of topics.
Distance
The distance metric between each paper and its citations.
# of Papers
The number of papers generated by the tool. Select a number between 1-50; the default is 20.
